# Personalized Networking Assistant

## Description

This project is developed as part of the Skill Wallet Internship Program.

The application helps users generate AI-powered networking conversation starters based on event information.

## Technologies Used

- Python
- Streamlit
- FastAPI
- Hugging Face Transformers
- DistilBERT
- GPT-2
- Wikipedia API
